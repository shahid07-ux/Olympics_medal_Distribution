# 🏅 Tokyo Olympics 2021 – Medal Analysis

## 🌍 Project Overview

This project explores and visualizes medal distribution from the **Tokyo 2021 Olympics** using Python.
It transforms raw CSV data into an insightful and easy-to-understand **grouped bar chart** comparing countries by their medal counts.

---

## 🎯 Objectives

* 📥 Read and process real-world Olympic data
* 🧹 Clean and organize dataset columns
* ➕ Compute total medals for each country
* 📊 Create a clear visual comparison of medal counts
* 🏆 Highlight top-performing nations

---

## 🚀 Key Features

* ✔️ Automatic data cleaning (removes unwanted spaces in column names)
* ✔️ Dynamic **Total Medal Calculation**
* ✔️ Sorting countries by performance
* ✔️ Grouped bar chart visualization (Gold 🥇, Silver 🥈, Bronze 🥉)
* ✔️ Clean and readable graph output

---

## 🛠️ Tech Stack

* **Python 🐍**
* **Pandas 📊** – Data handling
* **NumPy 🔢** – Numerical operations
* **Matplotlib 📈** – Data visualization
